# Glossary

| Term | Meaning |
| --- | --- |
| Buddy | Saayro's AI travel guide that helps users understand options and take structured actions |
| Trip Hub | the main trip workspace containing summary, itinerary, connected items, and exports |
| Connected Travel | optional trip context pulled from linked sources such as inbox or calendar |
| Handoff | opening or exporting trip context into another tool such as maps, notes, or messaging |
| Export Pack | a portable trip output, usually PDF or structured summary |
| Itinerary Stop | a single destination, activity, or transit anchor within a day plan |
| Travel Card | a reusable UI card representing a stop, booking, route, or recommendation |
| Preferred Maps App | the user's chosen map destination for route handoffs |
| Confidence Label | a visible cue describing how certain Saayro is about a recommendation or interpretation |
