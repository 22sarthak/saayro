# Saayro Product Brief

## Summary
Saayro is a premium pan-India AI travel companion that helps users plan, navigate, understand, organize, and manage trips from planning to return.

## Core proposition
Saayro is not just a trip planner. It is a travel operating layer with an interactive multimodal Buddy.

## Audience
- solo travelers
- couples
- family travelers
- group travelers
- premium convenience-seeking users

## Key differentiators
- multimodal Buddy
- connected travel from inbox and calendar over time
- open ecosystem exports and map handoffs
- premium, calm interface

## Value delivered
- centralizes trip intent, plans, and travel context
- reduces cross-tool chaos
- improves confidence through explainable suggestions
- supports sharing and handoff without lock-in

## Current phase
Front-end shell with mock data first.
