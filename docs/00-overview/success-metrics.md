# Success Metrics

## Activation metrics
- trip created in first session
- onboarding completed or intentionally skipped
- first Buddy interaction after trip creation

## Retention metrics
- week 1 retained users
- month 1 retained planners
- second-trip creation rate

## Engagement metrics
- itinerary edits per active trip
- Buddy sessions per active trip
- trip hub revisit rate

## Export and share metrics
- export_clicked rate
- share link creation rate
- notes and WhatsApp copy usage

## Connected-travel adoption metrics
- account_connected initiation rate
- connected account completion rate
- connected items attached to a trip

## Buddy helpfulness metrics
- helpfulness score after key interactions
- action completion after Buddy suggestion
- correction or hallucination report rate

## Reliability target
- crash-free session target: 99.5% or better in beta, 99.8% target before broad launch
