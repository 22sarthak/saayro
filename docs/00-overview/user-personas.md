# User Personas

## Solo explorer
- Goals: move quickly, discover confidently, avoid planning overload
- Frustrations: too much research, unclear routes, weak local context
- Preferred devices: mobile first, web for deeper planning
- Behavior patterns: spontaneous changes, frequent map use, quick queries
- Likely feature usage: Buddy, route previews, itinerary optimization, exports

## Couple planner
- Goals: align preferences, pace the trip well, reduce decision fatigue
- Frustrations: duplicated research, unresolved tradeoffs, unclear options
- Preferred devices: web for planning, mobile during travel
- Behavior patterns: compare options, save favorites, share often
- Likely feature usage: planner workspace, Buddy suggestions, shareable itineraries

## Family coordinator
- Goals: keep logistics predictable, reduce stress, protect timing buffers
- Frustrations: missed connections, too much coordination, comfort tradeoffs
- Preferred devices: both web and mobile
- Behavior patterns: plans ahead, checks status repeatedly, prefers structure
- Likely feature usage: trip hub, connected travel, export packs, detailed itinerary

## Friend-group organizer
- Goals: coordinate decisions fast, keep everyone aligned, share clear plans
- Frustrations: scattered chats, slow consensus, forgotten details
- Preferred devices: mobile for sharing, web for organizing
- Behavior patterns: iterative planning, lots of messaging, collaborative changes
- Likely feature usage: share links, WhatsApp export, trip summary cards

## Premium convenience traveler
- Goals: save time, stay informed, feel taken care of
- Frustrations: noisy interfaces, generic recommendations, unreliable AI
- Preferred devices: both web and mobile
- Behavior patterns: expects polish, values speed, prefers curated guidance
- Likely feature usage: Buddy, connected travel, premium presentation, smart summaries
