# Content And Tone

## Voice attributes
- calm
- clear
- competent
- warm
- concise
- travel-aware

## Buddy response tone
- sound like a helpful travel guide
- explain tradeoffs plainly
- show confidence carefully
- be brief in action moments and slightly richer in planning moments

## CTA copy style
- use direct labels such as `Build itinerary`, `Open in Maps`, `Connect account`
- avoid vague labels unless the context is obvious

## Error message tone
- explain what happened
- explain what the user can do next
- avoid blame or panic

## Permission request tone
- start with user benefit
- state scope and reversibility
- avoid legalistic wording in the first sentence

## Support copy style
- practical and respectful
- recovery-oriented
- no robotic templates
