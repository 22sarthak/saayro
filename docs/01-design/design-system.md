# Design System

## Colors
| Token family | Usage |
| --- | --- |
| Ivory | primary page surfaces |
| White | elevated cards and panels |
| Slate | typography, borders, and structure |
| Sky | maps, movement, flights |
| Violet | Buddy, intelligence, suggestions |
| Mint | success, connected status, confidence |
| Amber | discovery, food, warmth, culture |

## Typography
- editorial heading style with strong hierarchy
- highly legible sans-serif for body and UI
- use spacing and weight before color for hierarchy

## Spacing scale
- 4, 8, 12, 16, 24, 32, 48, 64
- prefer generous padding in cards and sections

## Radius scale
- small: 10
- medium: 16
- large: 24
- pill: full radius for chips and status badges

## Elevation rules
- soft shadows only
- prefer tonal separation over heavy floating effects

## Card patterns
- ivory base with white or softly tinted cards
- clear title, metadata, action area, and optional status row

## Status colors
- success: mint
- warning: amber
- info: sky
- Buddy insight: violet
- error: low-saturation warm red

## Icons and illustration guidance
- simple travel-relevant icons
- editorial landscape, route, and map motifs
- avoid neon or generic sci-fi visuals
