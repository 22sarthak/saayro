# Ivory Atlas System

## Purpose
Ivory Atlas is the approved visual direction for Saayro. It keeps the product bright, premium, and editorial while staying highly practical for travel use.

## Base surfaces
- primary background: warm ivory
- secondary background: soft white
- card surfaces: white, mist ivory, or lightly tinted accent wash

## Ivory and white layering
- use ivory as the main field
- float white cards and panels above it
- use tonal separation before borders
- reserve tinted sections for emphasis only

## Slate text hierarchy
| Level | Usage |
| --- | --- |
| Slate 950 | page headlines and major data |
| Slate 800 | body copy and card titles |
| Slate 600 | metadata and helper text |
| Slate 400 | dividers, placeholders, disabled treatment |

## Accent mapping
| Accent | Meaning |
| --- | --- |
| Sky | maps, movement, flights, route logic |
| Violet or lilac | Buddy AI, suggestions, reasoning |
| Mint or emerald | success, connected status, confidence |
| Amber | discovery, food, warmth, culture |

## Gradients allowed
- soft ivory to white gradients for hero moments
- light accent gradients only when they support a clear product meaning
- no dark gradients or neon glow treatments

## Shadow softness
- wide, low-opacity, warm-neutral shadows
- subtle depth rather than dramatic layering
- reduce shadow intensity on mobile

## Desktop vs mobile adaptation
- desktop: editorial spacing, multi-column planners, persistent context panels
- mobile: stacked cards, bottom sheets, thumb-reachable actions, preserved hierarchy

## Guardrails
- keep the interface bright
- use accent hues with meaning
- avoid dark AI styling
- never rely on color alone to convey state
