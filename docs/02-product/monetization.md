# Monetization

## Free tier
- create and manage trips
- use core Buddy interactions with sensible limits
- access basic exports and map handoffs

## Pro tier
- higher usage limits
- richer export packs
- advanced connected travel workflows
- premium trip organization features

## Possible affiliate layer
- consider later booking referrals or partner links
- never let affiliate incentives bias recommendations

## Connected-travel premium value
Connected travel should become a premium convenience layer only after quality, consent, and trust framing are mature.

## What should not be monetized early
- essential trust features
- basic portability features
- correction and recovery workflows
