# Buddy Interaction Flow

## Ask, respond, suggest, act loop
1. user asks a travel question
2. Buddy responds with concise, contextual guidance
3. Buddy may offer structured suggestions
4. user can convert a suggestion into an action such as adding a stop or refining a day

## Contextual prompting rules
- active trip context is default
- Buddy should clarify missing essentials only when necessary
- prompts should reference current destination, dates, and pace where available

## Escalation to structured actions
- add to itinerary
- refine a day plan
- compare options
- open route handoff
- draft export summary
