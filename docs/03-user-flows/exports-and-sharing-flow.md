# Exports And Sharing Flow

## PDF export
1. user chooses export
2. app shows format options
3. export is generated with itinerary, summary, and key travel details
4. success state offers open, download, or share

## WhatsApp share
- generate compact text blocks optimized for messaging
- preserve readability and clear day structure

## Notes copy
- copy a clean plain-text or rich-text itinerary summary
- avoid dumping raw metadata

## Shareable link
- create a share-ready trip summary view
- clearly indicate whether the link is view-only and whether it expires later

## Route share
- share a selected itinerary stop or route into a maps destination or messaging flow
