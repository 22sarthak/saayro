# Maps Handoff Flow

## Choose preferred maps app
- user selects preferred maps app in onboarding or settings
- preference should be editable anytime

## Open in Google Maps
- hand off a selected stop, route, or day sequence to Google Maps when available

## Open in Apple Maps
- hand off a selected stop or route to Apple Maps when available

## Use in-app preview
- show a lightweight route preview when external handoff is not the immediate choice

## Fallback when external maps are unavailable
- provide address copy, coordinates where available, and clear fallback messaging
