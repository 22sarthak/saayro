# Mobile App Spec

## Home
- recent trip summary
- quick create trip action
- active trip entry point

## Buddy
- chat-like surface with structured action chips
- voice entry later through explicit permission-gated controls

## Trip hub
- overview, itinerary, connected items, and exports in stacked sections

## Itinerary
- day-by-day card stack
- route preview and map handoff actions

## Maps
- preferred maps app actions and route preview state

## Profile
- preferences, maps app, connected states, and support entry points

## Permissions flow
- request permissions contextually and only when feature-relevant
