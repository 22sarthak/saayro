# State Management

## Server state
- trips
- itinerary data
- Buddy message history
- exports
- connected travel sources and items

## Local UI state
- sheet or modal visibility
- selected filters
- active day selection
- pending route preview state

## Persisted preferences
- preferred maps app
- onboarding choices
- lightweight display preferences

## Chat state
- current draft message
- streaming or pending response state later
- action chip context tied to active trip

## Selected trip context
- active trip should be globally accessible within authenticated app surfaces
