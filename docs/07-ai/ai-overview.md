# AI Overview

## Buddy role
Buddy is a travel guide embedded inside Saayro, responsible for clarifying options, structuring suggestions, and helping users act within trip context.

## Current capabilities
- contextual mock responses
- trip-aware suggestions
- structured action prompts

## Later capabilities
- grounded travel reasoning
- connected-travel-aware suggestions
- richer multimodal understanding

## Constraints
- Buddy must not overclaim freshness or certainty
- AI should remain travel-first and action-oriented
- complex model orchestration is not part of the current phase
