# Model Routing Plan

## Text tasks
- trip-aware dialogue
- summary generation
- itinerary refinement suggestions

## Image tasks
- interpretation of user-uploaded travel images
- extraction-oriented prompts where appropriate

## Speech tasks
- transcription first
- speech-native interaction only if it improves travel speed materially

## Generation vs extraction tasks
- use generation for guidance and summaries
- use extraction for connected travel and multimodal parsing where needed

## Cost tiers later
- define lightweight vs premium paths later
- advanced model orchestration is out of scope for the current phase
