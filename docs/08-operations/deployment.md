# Deployment

## Web deploy
- deploy Next.js app through a repeatable CI pipeline
- validate environment variables and public config before rollout

## Mobile build and distribution
- use Expo build and distribution flows appropriate to alpha, beta, and store release phases

## Backend deploy
- deploy FastAPI service with managed PostgreSQL and background worker support

## Secret handling
- keep secrets in environment-managed secret stores
- never commit secret material or embed provider credentials in client apps
