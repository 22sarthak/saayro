# QA Strategy

## Unit testing
- critical formatters, validators, and state helpers

## Integration testing
- trip creation, itinerary generation, auth entry, and export flows

## Manual testing
- end-to-end checks on major user journeys and state transitions

## Visual QA
- design token consistency
- responsive layout checks
- empty, loading, error, and partial states

## Device and browser coverage
- define minimum browser coverage for web
- define representative iOS and Android coverage for mobile
