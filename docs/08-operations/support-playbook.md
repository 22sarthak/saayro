# Support Playbook

## Failed OTP
- confirm number formatting
- retry with wait guidance
- escalate if provider outage suspected

## Export failed
- explain temporary failure
- offer retry and alternate share path if available

## Account connect failed
- explain whether the issue is consent, network, or provider-side
- suggest reconnect or later retry

## Wrong trip timing displayed
- distinguish manual data issue from connected import issue
- suggest editing or reattaching the item

## Broken map handoff
- confirm preferred app, fallback link, and address copy path
