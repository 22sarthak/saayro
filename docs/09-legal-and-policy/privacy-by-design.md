# Privacy By Design

## Minimal collection
Collect only the data needed to support trip planning, account access, and user-requested features.

## User control
Users should be able to review, disconnect, and remove connected features where supported.

## Transparency
Explain what is collected, why it is used, and whether it is stored or only processed temporarily.

## Connection scope limits
Request the narrowest reasonable provider scopes for auth, inbox access, or calendar access.
